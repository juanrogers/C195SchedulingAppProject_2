How to run application:
Please ensure that the correct driver isused for mySQL database connection. Open appication and login.
After successfully entering valid credentials, main screen for application will be displayed. From here, the user
can view customers, appointments, reports or logout of application.


From appointments screen - User can add, update and delete appointments. User can also view appointments by week, month or all.

From customers screen - User can add, update, and delete customers.

From reports screen - User can view reports for appointments by month and type, contact schedules, and number of customers.


Additional report for A3F:
The additional report will display the total number of customers current listed in database added by application's "add" customer function.
